# 💚 KeenKeeper

> Your personal shelf of meaningful connections. Browse, tend, and nurture the relationships that matter most.

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| **React.js** | UI framework |
| **React Router DOM** | Client-side routing (createBrowserRouter) |
| **Tailwind CSS v4** | Styling & responsive design |
| **Recharts** | Friendship analytics pie chart |
| **react-hot-toast** | Toast notifications |
| **Lucide React** | Icons |
| **Vite** | Build tool |

## ✨ Key Features

1. **Friend Management** — Browse friends in a 4-column responsive grid with color-coded status badges (On-Track, Almost Due, Overdue). Click any card for a full detail page.

2. **Quick Check-In with Live Timeline** — Log a Call, Text, or Video from any friend's detail page. Entries appear instantly on the Timeline with toast confirmations.

3. **Friendship Analytics** — A donut pie chart (Recharts) breaks down all your interactions by type, with live progress bars for each category.

## 🚀 Getting Started

```bash
npm install
npm run dev
```
